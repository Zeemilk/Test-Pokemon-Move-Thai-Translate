# 🎮 Pokemon Move Viewer

เว็บแอปแสดงรายละเอียดท่าต่อสู้ของโปเกมอน โดยโหลดข้อมูลจากไฟล์ `moves_th.json` ภายในโปรเจค

## ✨ คุณสมบัติ

- 🔍 **ค้นหาท่าต่อสู้** ด้วยชื่อท่าเป็นภาษาอังกฤษ
- 📊 **แสดงข้อมูลสำคัญ** เช่น Power, Accuracy, PP, Type และประเภทความเสียหาย
- 🌐 **คำอธิบายภาษาไทย** หากข้อมูลในไฟล์มีคำแปลไทย
- 📱 **Responsive Design** ใช้งานได้ทั้งหน้าจอเดสก์ท็อปและมือถือ
- ⚡ **โหลดข้อมูลจากไฟล์ JSON** โดยไม่ต้องเรียก API ภายนอกสำหรับข้อมูลท่า

## 🚀 การใช้งาน

### วิธีรันโปรเจค

1. เปิด `index.html` ในเว็บเบราว์เซอร์โดยตรง

หรือ

2. ใช้ HTTP server เช่น Python:

```bash
python -m http.server 8000
```

แล้วเปิด `http://localhost:8000`

### วิธีค้นหา

1. พิมพ์ชื่อท่าต่อสู้ในช่องค้นหา
2. เลือกท่าจากรายการคำแนะนำ หรือพิมพ์ให้ตรงกับชื่อท่า
3. กดปุ่ม `🔍 แสดง`
4. ดูข้อมูลท่าที่แสดงด้านล่าง

## 📋 ข้อมูลที่แสดง

- **ชื่อท่า** - ชื่อท่าต่อสู้
- **Type** - ประเภทของท่า เช่น Fire, Water, Electric
- **Damage Class** - ประเภทความเสียหาย เช่น กายภาพ, พิเศษ, สถานะ
- **Power** - ความแรงของท่า
- **Accuracy** - ความแม่นยำ
- **PP** - จำนวนครั้งที่ใช้ได้
- **คำอธิบาย** - ข้อความคำอธิบายท่า (ถ้ามี)

## 🛠️ เทคโนโลยี

- **HTML5** - โครงสร้างหน้าเว็บ
- **CSS3** - การจัดรูปแบบและจัดวาง
- **JavaScript** - โหลดข้อมูลและจัดการการค้นหา
- **JSON** - ข้อมูลท่าจากไฟล์ `moves_th.json`
- **MyMemory API** - ใช้แปลข้อความเป็นไทยเมื่อจำเป็น

## 📦 โครงสร้างไฟล์

```
Poke/
├── index.html      # หน้าเว็บหลัก
├── moves_th.json   # ข้อมูลท่าของโปเกมอน
└── README.md       # เอกสารนี้
```

## ⚠️ หมายเหตุ

- ข้อมูลท่าจะถูกโหลดจากไฟล์ `moves_th.json`
- หากแอปไม่สามารถอ่านไฟล์ได้ ให้ตรวจสอบว่ารันจาก HTTP server หรือไฟล์มีอยู่ในโฟลเดอร์เดียวกัน
- การแปลภาษาไทยขึ้นอยู่กับการเรียก MyMemory API ซึ่งอาจมีข้อจำกัดการใช้
