# 🎮 Pokemon Move Viewer

ตัวแสดงท่าการต่อสู้ของโปเกมอน ที่ช่วยให้คุณค้นหาและดูรายละเอียดของท่าต่อสู้ทั้งหมด

## ✨ คุณสมบัติ

- 🔍 **ค้นหาท่าต่อสู้** - ค้นหาท่าต่อสู้จากชื่อด้วยการพิมพ์
- 📊 **ข้อมูลโดยละเอียด** - แสดงพลัง, Accuracy, PP, Type และอื่น ๆ
- 🌐 **แปลเป็นไทย** - รายละเอียดแสดงผลเป็นภาษาไทย
- 📱 **Responsive Design** - ใช้งานได้ดีบนทุกอุปกรณ์
- ⚡ **การโหลดบทต่อทั้งหมด** - โหลด 1000+ ท่าของโปเกมอน

## 🚀 การใช้งาน

### วิธีการเปิดhostLocal:

```bash
# ใช้ Python HTTP Server
python -m http.server 8000
```

หรือเปิด `index.html` ตรงในเบราว์เซอร์

แล้วเข้าที่ `http://localhost:8000`

### วิธีการค้นหา:

1. พิมพ์ชื่อท่าต่อสู้ (เช่น `Tackle`, `Flame Burst`, `Thunderbolt`)
2. กดปุ่ม 🔍 **แสดง** หรือกด Enter
3. ดูรายละเอียดของท่านั้น ๆ

## 📋 ข้อมูลที่แสดง

- **ชื่อท่า** - ชื่อท่าต่อสู้
- **Type** - ประเภทของท่า (Fire, Water, Electric, เป็นต้น)
- **Power** - ค่าพลังของท่า
- **Accuracy** - ความแม่นยำ
- **PP** - Total Power Points (จำนวนครั้งที่ใช้ได้)
- **Effect** - คำอธิบายเอฟเฟกต์ของท่า
- **Priority** - ลำดับความสำคัญในการใช้ท่า

## 🛠️ เทคโนโลยี

- **HTML5** - โครงสร้างหน้าเว็บ
- **CSS3** - ออกแบบและ Responsive Layout
- **JavaScript** - ตรรกะและการทำงาน
- **PokéAPI** - ดึงข้อมูลท่าต่อสู้ของโปเกมอน
- **MyMemory API** - แปลข้อมูลเป็นภาษาไทย

## 📦 ไฟล์โครงการ

```
Poke/
├── index.html      # ไฟล์หลัก (HTML + CSS + JavaScript)
└── README.md       # ไฟล์นี้
```

## ⚠️ ข้อคำแนะนำการใช้งาน

- ต้องมีการเชื่อมต่อ **Internet** เนื่องจากใช้ API ภายนอก
- ครั้งแรกที่รันจะใช้เวลาสักครู่เพื่อดึงข้อมูลท่าทั้งหมด
- อาจใช้เวลาประมาณ 5-10 วินาที ขึ้นอยู่กับความเร็วอินเทอร์เน็ต

## 🎨 ออกแบบ

- **Color Scheme**: ม่วงและน้ำเงิน (Gradient) ตามธีมโปเกมอน
- **UI/UX**: สมัยใหม่ด้วย Rounded Corners และ Smooth Animations
- **Accessibility**: รองรับการใช้ Keyboard และ Screen Readers
